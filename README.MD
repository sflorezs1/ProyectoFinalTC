# Instrucciones de Python
Para usar la applicación de Python activar el entorno virtual e instalar los requisitos.  

` cd Python/RGBRemote `  
` ./bin/activate `  
` pip install -r requirements.txt `  
` python main.py `

Y acceder al url generado por Flask en un navegador.
